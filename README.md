# Weather App

This is a simple Weather App that allows users to search for any city and view real-time weather information.


## Features

- Search weather by city name

- Displays temperature, humidity, and weather conditions

- Real-time data fetched from a weather API

- Responsive design


## Technologies Used

- HTML

- JavaScript / React 

- Tailwind CSS

- API integration
